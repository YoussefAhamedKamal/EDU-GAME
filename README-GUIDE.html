دليل إضافة الشرائح في Genially

الطريقة 1: تضمين كـ iframe (الأسهل والأفضل)
  1. 1 انشر المشروع على Vercel بالضغط على Publish
  2. 2 انسخ رابط الموقع المنشور (مثل: https://your-project.vercel.app)
  3. 3 في Genially، اختر Insert → Others → External content
  4. 4 الصق الرابط واضبط حجم الإطار ليملأ الشريحة
الطريقة 2: تضمين HTML مباشرة في Genially
  1. 1 في Genially، اختر Insert → Others → Embed
  2. 2 انسخ كود HTML من أحد الملفات أدناه
  3. 3 الصق الكود في مربع التضمين
  4. 4 كرر لكل شريحة
ملاحظة: بعض ميزات JavaScript التفاعلية قد لا تعمل في وضع التضمين المباشر. الطريقة 1 مضمونة أكثر.
الطريقة 3: استضافة كل شريحة على حدة
  1. 1 ارفع ملفات HTML على GitHub Pages أو Netlify أو أي استضافة مجانية
  2. 2 كل ملف سيكون له رابط خاص
  3. 3 استخدم روابط الملفات الفردية في Genially
