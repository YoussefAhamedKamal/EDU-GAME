# اللعبة التعليمية جنيلي - حارس الأمان الرقمي

**وصف المشروع:**
اللعبة التعليمية التفاعلية تعلم الطلاب أساسيات الأمن السيبراني والحماية الرقمية من خلال 12 شريحة تعليمية و5 أسئلة اختبار.

## محتويات المشروع

### 📚 الشرائح التعليمية (12 شريحة)
- **slide-01-intro.html** - المقدمة
- **slide-02-what-is-cybersecurity.html** - ما هو الأمن السيبراني
- **slide-03-passwords.html** - كلمات المرور القوية (تفاعلي)
- **slide-04-phishing.html** - التصيد الاحتيالي
- **slide-05-privacy.html** - حماية الخصوصية
- **slide-06-wifi.html** - أمان شبكة الواي فاي
- **slide-07-updates.html** - التحديثات الأمنية (تفاعلي)
- **slide-08-social-engineering.html** - الهندسة الاجتماعية
- **slide-09-phone-security.html** - أمان الهاتف الذكي
- **slide-10-cyberbullying.html** - التنمر الإلكتروني
- **slide-11-backup.html** - النسخ الاحتياطي للبيانات
- **slide-12-summary.html** - ملخص الرحلة

### 🎮 أسئلة الاختبار (5 أسئلة)
- **quiz-01.html** - سؤال حول كلمات المرور
- **quiz-02.html** - سؤال حول التصيد الاحتيالي
- **quiz-03.html** - سؤال حول أمان الواي فاي
- **quiz-04.html** - سؤال حول التنمر الإلكتروني
- **quiz-05.html** - سؤال حول النسخ الاحتياطي

## كيفية الاستخدام

### الطريقة 1: تشغيل محلي
```bash
# انسخ المستودع
git clone https://github.com/YoussefAhamedKamal/EDU-GAME.git

# افتح index.html في المتصفح
```

### الطريقة 2: نشر على GitHub Pages
1. اذهب إلى Settings
2. اختر Pages
3. اختر main branch
4. سيكون الموقع متاح على: https://YoussefAhamedKamal.github.io/EDU-GAME/

### الطريقة 3: التضمين في Genially
1. استضف المشروع على Vercel أو أي منصة أخرى
2. استخدم الرابط للتضمين في Genially كـ iframe
